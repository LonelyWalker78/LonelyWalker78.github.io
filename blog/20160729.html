CSS的一些小技巧（不断更新中...）

前言

CSS可谓是博大精深，在开发过程中经常会遇到一些问题，谨用此文来记录本人所遇到过的一些问题，以便日后查阅

1、input标签在手机上点击无法获得焦点：没有设置type="text"。

2、弹性盒模型
    （1） 安卓4.1上内容缩小靠左，使用下面属性水平居中
      .f-hc{-webkit-box-pack:center;-webkit-justify-content:center;}/*水平居中*/
    （2） 弹性盒子模型div块因为文字大小而被撑开，没有平均分配：设置子元素width:0%,让宽度由 box-flex 属性来分配。

3、DIV内图片文字水平对齐
    （1） 图片和文字各放入独立的块级容器。然后以padding-top或margin-top来控制。
    （2） 对图片添加样式vertical-align:middle;如：<div><img src="url" style="vertical-align:middle;" />文字</div>

4、把图片作为背景：如果你的图片只是用来作为小图标放在文字的左侧，那就推荐用这个方法，图片设置成文字的背景，不循环，定位在左侧上下居中，文字向左padding图片的宽度加几个像素。
下面是示例代码：

<style> a img { border:none } .testTXT{ height:60px; line-height:60px; padding-left:65px; background:url(http://www.zc144.com/download/Template.jpg) no-repeat left center; } </style> <div class="testdiv"> <div class="testTXT"> <span>这里是文字,看看文字对齐了没</span> </div> </div>
